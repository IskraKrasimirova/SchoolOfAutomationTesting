# C# Fundamentals Programming Exam, 15.12.2025  

**Duration:** 2 hours  
**Difficulty:** Beginner → Intermediate  

**Allowed:** Primitives, arrays, collections, loops, conditionals, simple methods  
**Not Allowed:** OOP features (classes beyond Program), LINQ, library shortcuts (like Array.Reverse, Math.Sqrt, etc.)  

## General Rules
1. Use only primitive data types (bool, char, int, long, float, double, decimal, etc.) and arrays and collections.  
2. Input and output must use the console (`Console.ReadLine()`, `Console.WriteLine()`).  
3. Avoid using library functions that solve the task automatically.  
4. Code should be clear, modular, and properly indented.  
5. Comments are encouraged but not required.
